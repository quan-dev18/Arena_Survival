**# Arena_Survival**

A fun survival game on a platform filled with enemies, created by **QuanDev**.

### About the Project
This is a simple survival game where you must stay alive as long as possible while facing waves of enemies around you. The project was built primarily for learning game development and practicing programming skills.

> **Note**: This is a personal homework/project and may be deleted in the future.

### Features
- Survival gameplay on a platform arena
- Multiple enemies trying to take you down
- Built as a learning project
